# Repository Guide — Interaction Framework

This marks the finalized interaction structure for the Dynamo Adaptive System.
All contributions must follow continuity-safe handling rules.

---

## Valid Actions
- reference public documentation
- fork for observational learning
- cite original work
- use system output respectfully

---

## Restricted Actions
- altering continuity strings
- renaming structural components
- removing attribution tags
- redistributing modified copies

---

## Use-Tier Enforcement
System access operates under:
Public Evaluation Layer (view-only, non-active)

Modification rights:
Prohibited without ownership authority.
