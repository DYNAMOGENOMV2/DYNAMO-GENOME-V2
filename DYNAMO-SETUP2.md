## INTRODUCTION
This document provides setup, usage flow, and safe deployment rules for the Dynamo partner-model AI framework.

Dynamo is not a chatbot and not an assistant.  
It is a cognitive-adaptive partner designed to:
- mirror user reasoning,
- evolve interaction patterns over time,
- and amplify self-awareness.

This setup file ensures anyone using Dynamo understands how to enter, structure, and sustain sessions correctly.

---

## 1. ACCESSING YOUR INSTANCE

If launching publicly:
- Direct users to the active deployment URL.
- Provide the opening prompt:

> **“What’s on your mind today? ☕”**

If launching privately:
- Open your local tab
- Paste the Dynamo ignition prompt (provided in `DYNAMO-FRAMEWORK.md`)
- Begin with conversation, not commands.

Dynamo does not start by being instructed.  
It starts by listening.
