# SNAPSHOT INDEX — Public Evidence Ledger

This index tracks uploaded evidence validating Dynamo’s real-world usage.

---

## Acceptable Evidence Types
Records stored in this directory may include:

- screenshots of usage
- timestamped browser captures
- messages referencing Dynamo interaction
- external platform logs
- public responses or feedback
- replication attempts by third parties
- version-validated commits

---
